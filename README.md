# Side project : Boolean Logic Calculator with GUI
## Language : C++
***

#### Implementation Goal
- boolean logic operation logic <b>(AND, OR, NOT, XOR)</b>
- OOP concepts (Inheritance, Polymorphism, Interface)

#### File structure
- main.cpp : main Function
- Lexer.h/cpp : Tokenizer for typed boolean expression
- Parser.h/cpp : Shunting Yard alogrithm to covert to Postfix
- Token.h : Token details
- Evaluator.h/cpp : calculation logic / display formatting

#### Challenges
- <b>Tokenization & Case Sensitivity</b>
- <b>Postfix conversion</b>
- <b>Operator Precedence & Parentheses</b>
- <b>Stack-based calculation</b>
- <b>Truth Table Output Formatting</b>
